# Unity Junior Programmer Practice Project Mission One
## Driving and Plane simulator

This project is part of my Unity learning course and was developed while following the Unity Junior Programmer Pathway.
This pathway teaches beginner-to-intermediate Unity programming concepts with hands-on projects and missions. It’s designed for learners who want to:
- learn to code in C# within Unity,
- understand core development workflows, and
- prepare for entry-level roles in game development or real-time 3D applications.

### Communication & Collaboration
- Participating in feedback loops and learning teamwork practices.
